HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The meta and title element are located inside the head element.
  • HTML attributes are never the closing element, always the beginning.
  • HTML adds structure to the content of a webpage.
    •

    CSS

  • CSS adds style to a webpage.
  • Defines the colors, aesthetics, visual layouts, and organization.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Three styles: inline CSS, internal CSS style sheet, and external CSS style sheet.
  • Inline CSS styles an element directly in the HTML file

    Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git add -A commits all files in the working branch.
  • git commit -m associates a message with our commit.
